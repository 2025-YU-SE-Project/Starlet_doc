### 📌 문서 저장소 브랜치 전략


* **`main`**: 프로젝트를 볼 수 있는 방문자들이 해당 문서를 읽을 수 있도록 하는 기본 브랜치입니다.
* **`document/`**: 문서에 대한 작업을 할때 생성하는 브랜치 입니다.
